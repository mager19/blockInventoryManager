Description
Easily Inventory Gutenberg Blocks = Block Inventory is a developer tool that provides a user interface to inspect Gutenberg blocks across your WordPress site.

Features of Block Inspector =
Scans all pages and post types to identify used blocks
Provides an option to search for a particular block and shows on which pages or custom post types it is used
Settings options allow to set a block prefix and an option to set a transient
Usage
After activating the plugin, go to BlockInventory. Here you can see all the blocks used across your site. You can also search for a particular block to see where it is used.
